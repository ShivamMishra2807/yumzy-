Yumzy - Online Food Ordering System

Yumzy is a simple online food ordering web application built as a college project to learn full-stack web development. The idea behind this project was to create a platform where users can browse different food items, search for their favorite dishes, add them to a cart, and place orders through an easy-to-use interface.

While developing this project, we worked with both frontend and backend technologies. The frontend was created using HTML, CSS, and JavaScript to build a responsive and interactive interface, while the backend was developed using Python and Flask. SQLite was used to store the application's data. This project helped us understand how different parts of a web application work together and gave us practical experience with database integration and version control using Git and GitHub.

Features

- Responsive and clean user interface
- Browse available food items
- Search for food items
- Add and remove items from the cart
- Place food orders

Getting Started

Clone the repository

```bash
git clone https://github.com/ShivamMishra2807/yumzy-.git
```

Move into the project directory

```bash
cd yumzy-
```

Install the required package

```bash
pip install flask
```

Run the application

```bash
python app.py
```

After running the server, open your browser and visit

```text
http://127.0.0.1:5000
```
